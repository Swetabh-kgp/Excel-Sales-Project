# Excel Sales Performance Dashboard
## Project Overview
This project is an interactive Sales Performance Dashboard built in Microsoft Excel.  
The dashboard helps analyze sales executive performance across different cities.
## Dashboard Features
- City-wise filtering using slicers
- High sales and low sales analysis
- Target hit percentage tracking
- Away from target percentage analysis
- Sales executive performance comparison
- Interactive charts and summary tables
## KPIs Used
- Total Sales
- Total Sales High
- Total Sales Low
- Target Hit Percentage
- Away From Target Percentage
## Charts Used
- Horizontal bar chart
- Pie chart
- Line chart
- Pivot tables
## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Formatting
- Conditional Formatting
- Data Analysis
## Purpose of the Dashboard
The purpose of this dashboard is to track sales performance, compare sales executives, identify top and low performers, and monitor target achievement in a simple visual format.
